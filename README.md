# Fabled Lands – Character & Progress Tracker

An online character and progress tracker for the **Fabled Lands** gamebook series, hosted as a static site on GitHub Pages.

## About

Fabled Lands is a series of open-world gamebooks where players explore a vast fantasy world, making choices that shape their adventure. This tracker helps players manage their characters and record their progress across the books — all from the browser.

## Features

- **Character Management** – Create and manage your Fabled Lands characters, including stats, abilities, and inventory.
- **Progress Tracking** – Keep track of visited locations, completed quests, and key decisions.
- **Local Storage** – All data is stored locally on your device using the browser's local storage, so your progress persists between sessions without the need for an account or server.
- **Static Site** – Hosted entirely on GitHub Pages with no backend required.

### Development

```bash
npm install
npm run dev
```

### Build for Production

```bash
npm run build
```

The static output in `dist/` can be deployed directly to GitHub Pages.

## Privacy

Your data never leaves your device. Everything is saved in your browser's local storage — no accounts, no servers, no tracking.
